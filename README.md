# DraftMind

Утилита для чтения верхней части экрана драфта Dota 2 и распознавания выбранных героев.

## Что уже есть

- захват верхней области экрана через `mss`
- выделение слотов Radiant/Dire из области драфта
- распознавание героя по шаблонам OpenCV
- структурированный результат, который можно потом отправлять в STRATZ

## Запуск

Анализ тестового скриншота:

```bash
python main.py --image screenshot.png
```

Анализ верхней части экрана:

```bash
python main.py --capture
```

## Рекомендации по пикам через STRATZ

Для рекомендаций нужен токен STRATZ в переменной окружения `STRATZ_API_TOKEN`.

```bash
export STRATZ_API_TOKEN=ваш_токен
python main.py --image screenshot.png --suggest --top 5
```

В ответе появится блок `suggestions` с:

- `recommended` — лучшие варианты для команды по текущему драфту
- `fills` — предложенное заполнение свободных слотов по одному
- `open_slots` — свободные слоты команды
